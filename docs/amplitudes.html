The state is the table

No Bloch sphere here. A quantum state is literally one arrow per outcome. We write each as √(prob%) ↺ phase — the same √% ↺ arrows the Complex page ends on: the length is √(probability) so the chance reads right off, and the ↺ phase turns it anti-clockwise from the +1 axis. Gates are small local rules that mix and cancel arrows. Entanglement is when the table refuses to split into separate qubits.

Things to try. Hit Bell recipe: watch two arrows appear and the table become un-splittable. Hit Interference (H, Z, H on one qubit): the |0⟩ arrow grows, then its two contributions point opposite and cancel to zero — that destructive cancellation, impossible for probabilities, is the entire quantum edge. Flip qubits 1→2→3 and watch the table double each time: that doubling is the exponential state space, in the flesh. Finally, pick on qubit then hover H: the table colours into the pairs that gate mixes — the same (A+B, A−B)/√2 two-arrow move on every pair that differs only in that bit, blind to all the others. That locality is why one little rule drives an exponential table.
