The state is the table

No Bloch sphere here. A quantum state is literally one arrow per outcome. We write each as √(prob%) ↺ phase — the same √% ↺ arrows the Complex page ends on: the length is √(probability) so the chance reads right off, and the ↺ phase turns it anti-clockwise from the +1 axis. Gates are small local rules that mix and cancel arrows. Entanglement is when the table refuses to split into separate qubits.

The same state, as a Bloch sphere

The very same amplitudes, drawn the friendly way — each arrow is that qubit's own state. For two qubits the centre coupling sphere carries their correlation: hit Bell recipe and the arrows blank out while the coupling ball lights up — the entanglement the spheres alone can't hold. Past 2 qubits even that fails; the table is all you've got. Drag to rotate.

One more to try. Pick a target with on qubit, then hover H: the table colours into the pairs that gate mixes — the same (A+B, A−B)/√2 two-arrow move on every pair that differs only in that bit, blind to all the others. That locality is why one little rule drives an exponential table.
