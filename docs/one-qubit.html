Why "complete"?

For one qubit the arrow is the entire state: a pure state always sits on the surface (|r⃗| = 1), and its direction (θ, φ) fixes the two amplitudes. The only thing not drawn is the global phase — and that is physically unobservable. Nothing is hidden. Add a second qubit and that stops being true.
Gate dictionary — what each does to the arrow

Every single-qubit gate is a rotation of the Bloch sphere. Hover any button to see its axis and turn angle drawn on the sphere.

  • I — identity: no rotation, the arrow doesn't move.
  • X / Y / Z — 180° about the x / y / z axis. X and Y flip |0⟩↔|1⟩ (north↔south); Z is a phase flip (|+⟩↔|−⟩), leaving |0⟩,|1⟩ put.
  • H — 180° about the diagonal (x+z)/√2; swaps the z and x axes (|0⟩↔|+⟩, |1⟩↔|−⟩).
  • S, S† — ±90° about z (S = √Z). T, T† — ±45° about z (T = √S = ⁴√Z). The †-versions turn the other way.
  • √X, √X† — ±90° about x (√X = "half a NOT": apply it twice to get X).
  • P(θ) — phase gate, θ about z. Same arrow motion as Rz (they differ only by an invisible global phase); P(π)=Z, P(π/2)=S, P(π/4)=T.
  • Rx, Ry, Rz (θ) — a tunable rotation by θ about x / y / z.
  • U(θ,φ,λ) — the universal gate = Rz(φ)·Ry(θ)·Rz(λ); it reaches any orientation, so every gate above is a special case.
